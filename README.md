# Parsero
Parsequinho from scratch

### Setup
- **Nix**
```bash
nix-build
cabal --enable-nix repl lib:parsero
```

- **Stack**
```bash
stack install
stack repl
```

- **Cabal new-\***
```bash
cabal new-update
cabal new-repl
```
